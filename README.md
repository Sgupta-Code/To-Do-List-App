# **To-Do List App**

Welcome to the **To-Do List App** repository! This simple yet effective application helps you organize your tasks efficiently, ensuring you stay on top of your daily activities.

## **Features**

- **Add Tasks:** Easily add tasks to your to-do list with a title and optional description.
- **Mark as Completed:** Check off tasks as you complete them to keep track of your progress.
- **Delete Tasks:** Remove tasks from your list when they're no longer needed.
- **Filtering:** Filter tasks by their completion status to focus on what needs to be done.

## **Technologies Used**

- **Frontend:** HTML, CSS, JavaScript
- **Version Control:** Git

## **Installation**

To run this application locally, follow these steps:

1. Clone this repository to your local machine using `git clone https://github.com/Sgupta-Code/to-do-list.git`
2. Navigate to the project directory: `cd to-do-List-App`
3. Install the dependencies using npm: `npm install`
4. Set up a MongoDB database and update the connection details in `config.js`
5. Start the server: `npm start`
6. Visit `http://localhost:3000` in your web browser to use the **To-Do List App**.

## **Usage**

- **Adding Tasks:** Type your task title and description (optional) into the input fields and press Enter or click the "Add Task" button.
- **Marking Tasks as Completed:** Click the checkbox next to a task to mark it as completed.
- **Deleting Tasks:** Click the delete button (trash can icon) next to a task to remove it from your list.
- **Filtering Tasks:** Use the filter options at the top of the list to show all tasks, only completed tasks, or only incomplete tasks.
